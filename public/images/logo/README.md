# Logo Storage

Place your logo files in this folder. Recommended formats:

- **Logo file**: `logo.png` or `logo.svg` (recommended for web - scalable and smaller size)
- **Logo with text**: `logo-text.svg` or `logo-text.png`
- **Favicon**: Place in `/public/` (already configured)

## File Naming Convention

- `logo.svg` - Main logo
- `logo-dark.svg` - Dark version (if needed)
- `logo-light.svg` - Light version (if needed)
- `logo-icon.svg` - Icon-only version

## Usage in Website

The logo will be referenced in the header component as:
```html
<img src="/images/logo/logo.svg" alt="Nutri Sofía Monroy" />
```
