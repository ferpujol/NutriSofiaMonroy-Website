# Nutri Sofía Monroy — Landing Page

A modern, elegant single-page landing website for a nutritionist specializing in **functional nutrition** with expertise in metabolic health and personalized nutrition.

Built with **Angular 19** and styled with **SCSS** using a **pastel green color palette**.

## Getting Started

```bash
npm install
ng serve
```

Navigate to `http://localhost:4200/`.

## Build

```bash
ng build
```

Build artifacts are stored in `dist/`.

## Project Structure

- `src/app/components/header/` — Fixed navigation with smooth scrolling
- `src/app/components/hero/` — Hero section with WhatsApp CTA
- `src/app/components/sobre-mi/` — About section with credentials
- `src/app/components/epigenetica/` — Educational section on epigenetics
- `src/app/components/redes-sociales/` — Social media links
- `src/app/components/agenda/` — WhatsApp appointment booking CTA
- `src/app/components/footer/` — Footer with copyright
- `src/styles/_variables.scss` — Shared SCSS color palette and typography variables
- `src/app/shared/config.ts` — Shared site configuration (WhatsApp URL)
